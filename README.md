# ⚡ HI-TECH ELECTRIC WORK

A modern, responsive full-stack business website for **HI-TECH ELECTRIC WORK**, providing professional electrical installation, maintenance, repair, and inverter services.

## 🌐 Features

- Responsive UI (Desktop, Tablet & Mobile)
- Modern Homepage
- About Page
- Services Page
- Individual Service Detail Pages
- Contact Page
- WhatsApp Floating Button
- Contact Form
- Email Notifications using Nodemailer
- Smooth Scroll Animations (AOS)
- SweetAlert2 Success & Error Messages

---

## 🛠 Tech Stack

### Frontend

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome
- AOS Animation Library
- SweetAlert2

### Backend

- Node.js
- Express.js
- Nodemailer
- dotenv

## ⚙ Environment Variables

Create a `.env` file inside the backend folder.

```
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password
PORT=5000
```

---

## 🚀 Run Locally

### 1. Clone Repository

```
git clone https://github.com/yourusername/hi-tech-electric.git
```

### 2. Install Backend

```
cd backend
npm install
```

### 3. Start Server

```
npm start
```

or

```
node server.js
```

### 4. Open Website

Open:

```
index.html
```

using Live Server.
